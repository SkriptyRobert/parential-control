# Format: AdBlock/AdGuard syntax
# Last updated: 2025-12-23

# Syntax examples:
#   ||domain.com^           - Block domain.com and all subdomains
#   @@||domain.com^         - Whitelist (allow) domain.com
#   ||*keyword*^            - Block any domain containing "keyword"
#   /regex/                 - Block using regular expression
#
# Documentation: https://adguard-dns.io/kb/general/dns-filtering-syntax/
